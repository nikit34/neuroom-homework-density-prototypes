# Настройка опроса «Экран заданий»

## Быстрый старт (без Google Sheets)

Просто открой `index.html` в браузере — ответы сохранятся в `localStorage`.

## С отправкой в Google Sheets

1. Создай новую таблицу: [sheets.new](https://sheets.new)
2. Создай листы: **Answers** и **Events**
3. Открой **Расширения → Apps Script**
4. Удали всё содержимое и вставь код из `google-apps-script.js`
5. Нажми **Развернуть → Новое развёртывание**
   - Тип: Веб-приложение
   - Выполнять от: Меня
   - Доступ: Все (включая анонимных)
6. Скопируй URL развёртывания
7. Вставь его в `index.html` в строку `GOOGLE_SCRIPT_URL = '...'`

## Структура опроса (чемпионат)

**Четвертьфиналы:**
- ЧФ1 — Простой список vs Канбан-доска
- ЧФ2 — Список по приоритету vs Календарь
- ЧФ3 — Таймлайн vs Таблица (дни x предметы)
- ЧФ4 — Quick Wins (bye, автопроход)

**Особые ответы:**
- «Оба нормально» — оба проходят, дополнительные матчи в полуфинале
- «Оба не понравились» — оба выбывают, соперник проходит без боя

**Полуфиналы** (динамически):
- ПФ1 — Прошедшие из ЧФ1 + ЧФ2 (1-4 участника)
- ПФ2 — Прошедшие из ЧФ3 + Quick Wins (1-3 участника)
- Последовательные 1v1 до определения финалиста

**Финал:**
- Финалист ПФ1 vs Финалист ПФ2

**Финальный экран:**
- Полное дерево турнира
- Кнопки шаринга (Telegram, VK)
- Форма заявки (Telegram-ник)
